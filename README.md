# Python--Customer-Churn-Project

Objective:
Analyze and predict customer churn for the telecom company "Neo" using data manipulation, visualization, and machine learning models.

Tasks:
Data Manipulation: Extracted specific columns, filtered records, and calculated counts.
Visualizations: Created bar-plots, histograms, scatter-plots, and box-plots.
Linear Regression: Predicted MonthlyCharges using tenure.
Logistic Regression: Predicted Churn based on MonthlyCharges and tenure.
Decision Tree: Predicted Churn based on tenure.
Random Forest: Predicted Churn using tenure and MonthlyCharges.

Technologies Used:
Python, Pandas, Matplotlib, Seaborn, Scikit-learn
